# 생성 패턴

생성 패턴은 객체 생성 및 인스턴스화를 처리하는 객체 생성 메커니즘에 중점을 둔다. 

이러한 패턴의 목적은 유연하고 유지 관리가 가능하며 효율적인 방식으로 개체를 만드는 방법을 제공한다.
