## 설명
기존 코드를 수정하지 않고 부가 기능을 추가하는 패턴 

데코레이터 패턴은 실제 구현을 담당하는 컴포넌트와 기능 확장을 위한 데코레이터들이 분리되는것이 핵심이다.
컴포넌트는 확장된 데코레이터들에게 위임을 통해 기능을 확장해 나간다.

따라서, 데코레이터를 조합하는 방식으로 기능 확장이 가능하다. 또한 기능 적용 순서도 손쉽게 변경 가능하다.
